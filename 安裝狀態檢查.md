## 🚀 下一步操作

### 步驟 1：安裝 Node.js

#### macOS 安裝方法

**方法 1：使用 Homebrew（推薦）**
```bash
# 如果尚未安裝 Homebrew，先執行：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 然後安裝 Node.js：
brew install node
```

**方法 2：從官網下載**
1. 前往 [https://nodejs.org/](https://nodejs.org/)
2. 下載 LTS（長期支援）版本
3. 執行安裝程式

**方法 3：使用 nvm**
```bash
# 安裝 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 重新載入終端
source ~/.zshrc

# 安裝 Node.js
nvm install --lts
nvm use --lts
```

#### Windows 安裝方法

**方法 1：從官網下載（推薦）**
1. 前往 [https://nodejs.org/](https://nodejs.org/)
2. 下載 LTS 版本
3. 執行安裝程式
4. **重要：** 安裝時請勾選「Add to PATH」選項

**方法 2：使用 Chocolatey**
```powershell
# 以系統管理員身份執行 PowerShell
choco install nodejs-lts -y
```

**方法 3：使用 nvm-windows**
1. 前往 [https://github.com/coreybutler/nvm-windows/releases](https://github.com/coreybutler/nvm-windows/releases)
2. 下載 `nvm-setup.exe` 並安裝
3. 在命令提示字元執行：
   ```cmd
   nvm install lts
   nvm use lts
   ```

### 步驟 2：驗證安裝

安裝完成後，**重新開啟終端/命令提示字元**，然後執行：

```bash
node --version
npm --version
```

應該會顯示版本號，例如：
```
v20.11.0
10.2.4
```

### 步驟 3：安裝專案依賴

在專案目錄下執行：

```bash
cd /Users/user/Documents/GitHub/tsa-repair-system
npm install
```

這會安裝所有必要的套件，可能需要幾分鐘時間。

### 步驟 4：啟動開發伺服器

**使用啟動腳本：**
- macOS/Linux: `./start.sh`
- Windows (CMD): `start.bat`
- Windows (PowerShell): `.\start.ps1`

**或直接使用 npm：**
```bash
npm run dev
```

然後在瀏覽器打開 `http://localhost:3000`
